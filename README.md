# Serene Observer

静默守护每一次构建，用数据勾勒代码仓库的心跳。

A lightweight CI health dashboard that watches build pipelines and surfaces
anomalies before they become incidents. Runs on GitHub Actions with hourly
checkpoints.

## Features
- Scheduled build status aggregation
- Drift detection across pipeline stages
- Automated daily summaries committed back to repo

> "不是为了发现问题，是为了确认一切安好。"

Built with Python. Data lives where code lives.
